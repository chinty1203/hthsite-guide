# hthsite-guide

## 项目简介

`hthsite-guide` 是一个用于归档和发布独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，仅用于存放和展示各类静态 HTML 文件。  
每个页面均为独立内容，适合作为示例、存档或轻量级信息展示页。

## 目录说明

```
/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 公共资源文件（如 CSS、JavaScript、图片等）
├── index.html      # 仓库主页（可选，用于导航或说明）
└── README.md       # 本文件
```

- `pages/` 目录下每个子目录或文件对应一个独立的 HTML 页面。
- `assets/` 目录存放页面共享的资源文件，避免重复引用。
- 根目录的 `index.html` 可作为入口页面，也可根据需求删除或替换。

## 页面归档说明

- 所有 HTML 页面均为静态文件，不依赖后端服务。
- 页面内容由各贡献者或归档者自行维护，仓库仅负责存储与分发。
- 归档的页面可能包含示例数据、演示内容或历史记录，不代表任何官方立场。
- 建议在页面头部添加版本号或归档日期，便于追溯。

## 维护说明

- 欢迎提交 Pull Request 添加或更新页面。
- 新增页面请放置在 `pages/` 下，并保持目录结构清晰。
- 涉及公共资源的修改，请同步更新所有引用该资源的页面。
- 仓库维护者保留对内容进行审核和管理的权利。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/hthsite-guide.git
   ```
2. 直接在浏览器中打开 `pages/` 下的 HTML 文件即可查看。
3. 也可通过 GitHub Pages 等服务部署后在线访问。
